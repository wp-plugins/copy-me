copy-me allows you to copy a post, page or custom post type to another site in a Wordpress multisite.

== Description ==

With copy-me you can copy any post, page or custom post type to another site in your Wordpress multisite.

== Installation ==

1. Upload copy-me to the `/wp-content/plugins/` directory
2. Go to the plugins page and activate the plugin through the 'Plugins' menu in WordPress
3. In the posts page you will find a new column called `Copy Me`.
4. Click the link in that column to copy the post/page/custom post type to the site of your choosing.
